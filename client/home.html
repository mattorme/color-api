Color Schemas

Colour palette generator and API
for the discerning developer


Get Started

Right to Randomise

Press -> to randomly generate a colour palette or <- to go back to previously viewed random colour palette

Lock Colors

Click to lock in colours that you like, while you randomise more colours for the palette. Useful for building a palette around a few known color themes

Save Palettes

Save color palettes to your profile once you have found a colour palette of your liking

Favourite Palettes

Find yourself using a particular palette often? Favourite the palette for ease of use

View and Explore

View previously saved palettes and explore palettes saved by other users

API available for export

Export your user color palettes in JSON format to your workspace using our cool API
